# Project Skills

Primary skills location for project bootstrap and agent workflows.

## Included

- `AI-Research-SKILLs/` (external repository, bootstrapped by script)
- `agent-instructions-generator/` (local skill to generate `CLAUDE.md` and `AGENTS.md`)

## Bootstrap external skills

Run:

```bash
./scripts/bootstrap-project-skills.sh
```

For another project:

```bash
./scripts/bootstrap-project-skills.sh /path/to/another/project
```
